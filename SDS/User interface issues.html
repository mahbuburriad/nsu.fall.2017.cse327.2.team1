4. User interface issues

The user interface of the Project will be divided into two main section: “PC clients” application and “Android clients” application. The PC client application divided into two major section “Admin Panel” and “User Panel”. This tool will allow the various users of accomplish the management of the software and the management of the rest of their projects. The SRS represent the type of people who will make use of the Project – and the purpose of this section is to describe how the user interface allow those people to do their tasks.
