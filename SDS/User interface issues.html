4. User interface issues

The user interface of the Project will be divided into two main section: “PC clients” application and “Android clients” application. The PC client application divided into two major section “Admin Panel” and “User Panel”. This tool will allow the various users of accomplish the management of the software and the management of the rest of their projects. The SRS represent the type of people who will make use of the Project – and the purpose of this section is to describe how the user interface allow those people to do their tasks.

4.1 Pc Clients

The Pc clients are divided into two section that is admin login option and login as Client. First page will be description about this project.

  • 4.1.1 Login as admin with PC

    • Login Screen- “Login” Option is found in Main page and all front-end of website. This login screen will allow to login in admin panel with rest of the application.

      After login, the application will present this screen and will list all of the projects that the client program knows are assigned to the current user. This may be incomplete or out of date if the user has not synchronized with the Project Server recently. The client will automatically attempt to synchronize after login. During this synchronization a potential conflict can occur if a project has been altered both server side and client side during the time the client was offline. The user will be asked to resolve such conflicts before being allowed to go onto other tasks. This screen includes:

      • Users
      • Home/Flat
      • Balance
      • Admin Control
      • Request (By users) Form
      • Registration

    • User Screen- This screen shows all users information (their username, email, location and other profile information. In this screen shows which user give ad or not. And shows the all users all history. Admin can edit, delete and have full control.

    • Home/Flat- This screen shows all data and information of running and non-running house sell/buy information.

    • Balance- This interface shows the total balance and all transaction. In this screen admin can see the specific user’s transaction.

    • Admin Control- This page shows all admin information, add new admin or employee, communication with one employee to another. Set the privilege and also can discard.

    • Request Form- This page shows the any kinds of request or any mail from user. Admin can see and after fix or resolve issues they can reply mail by automatically.

    • Registration: If one of user call to helpline and want to make their registration then admin can register with user’s information here.

4.1.2 Login as Client

  • Login Screen- “Login” Option is found in Main page and all front-end of website. This login screen will allow to login in admin panel with rest of the application.

    After login, the application will present this screen and will list all of the projects that the client program knows are assigned to the current user. This may be incomplete or out of date if the user has not synchronized with the Project Server recently. The client will automatically attempt to synchronize after login. During this synchronization a potential conflict can occur if a project has been altered both server side and client side during the time the client was offline. The user will be asked to resolve such conflicts before being allowed to go onto other tasks. This screen includes:

    • Profile
    • Home/Flat
    • Payment Method
    • Booking list
    • Contact Us
    • FAQ
    • Live Chat

  • Profile- Here user can see their profile information and if they want they can edit and pop-up dialog box shows the all information and how to edit. For so they can edit their profile directly by direction.

  • Home/Flat- This screen shows all data and information of running and non-running house sell/buy information. User can directly book their chosen home/flat with payment.

  • Payment Method- Here will several payment options. User can choose any option and they can add their card and all other information.

  • Booking List- This page shows the all information about their all booking and their list and information with table.

  • Contact Us- This page shows them how to contact automated accommodation office and also include other information about office. And also shows the direction from their location with google map.

  • FAQ- This page shows them all about the automated accommodation and all information with lots of instruction and Rules and privacy. And Some question’s answers.

  • Live Chat- This option will communicate with one another buyer and seller. And also with admin of their any problem and issues.
