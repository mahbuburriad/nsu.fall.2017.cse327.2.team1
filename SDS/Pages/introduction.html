1.0 Introduction

1.1 Purpose of this document:

The purpose of this Design Document is to present the system design of the software named Automated Accommodation System by which anybody can findout any suitable flat or house. Vacant flat or house which are stored in our database can be directly traced to the specific system objective along with providing more detailed data, functional, and technical requirements.This Design Document will verify that the current design meets all of the explicit requirements contained in the actual model as well as the necessary requirements desired by the customer. It also aims to give a more detailed understanding of the design and architecture of the Automated Accommodation System. Topics covered include the following:

  • Detailed description of components
  • User interface design
  • Architecture of the design
  • Processing narratives
  • Design constraints and restrictions
  • Test cases and expected results
